# Day 01 – AWS IAM Basics with Terraform

## What This Builds
This module demonstrates how to create and manage AWS IAM resources using Terraform,
including IAM users, policies, and policy attachments following least-privilege principles.

## AWS Services Used
- AWS IAM

## Resources Created
- IAM User
- IAM Policy
- IAM Policy Attachment

## Architecture
Terraform → AWS IAM → User & Policy Management

## How to Deploy
terraform init
terraform plan
terraform apply

## How to Clean Up
terraform destroy

## Key Learnings
- IAM user and policy structure
- Policy attachment best practices
- Importance of least privilege access
